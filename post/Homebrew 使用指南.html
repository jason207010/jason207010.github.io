安装Homebrew

获取安装脚本

1

  curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install > install

修改安装脚本源为中科大的源

1

  BREW_REPO = "https://github.com/Homebrew/brew".freeze

改为：

1

  BREW_REPO = "https://mirrors.ustc.edu.cn/brew.git".freeze

安装

1

  /usr/bin/ruby install

修改软件包源为中科大的源

Homebrew 源代码仓库

1
2

  cd "$(brew --repo)"
  git remote set-url origin https://mirrors.ustc.edu.cn/brew.git

重置：

1
2

  cd "$(brew --repo)"
  git remote set-url origin https://github.com/Homebrew/brew.git

Homebrew 核心软件仓库

1
2

  cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core"
  git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git

重置：

1
2

  cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core"
  git remote set-url origin https://github.com/Homebrew/homebrew-core

Homebrew cask 软件仓库，提供 macOS 应用和大型二进制文件

1
2

  cd "$(brew --repo)"/Library/Taps/homebrew/homebrew-cask
  git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-cask.git

重置：

1
2

  cd "$(brew --repo)"/Library/Taps/homebrew/homebrew-cask
  git remote set-url origin https://github.com/Homebrew/homebrew-cask

Homebrew 预编译二进制软件包

1
2

  echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.bash_profile
  source ~/.bash_profile

常用命令

命令 释义 示例
brew search [TEXT /REGEX/] 搜索软件包 brew search redis
brew info [FORMULA…] 显示软件内容信息 brew info redis
brew install FORMULA… 安装软件 brew install redis
brew uninstall FORMULA… 卸载软件 brew uninstall redis
brew reinstall [options] formula 重新安装软件 brew reinstall redis
brew upgrade [FORMULA…] 升级软件 brew upgrade redis
brew list [FORMULA…] 显示已经安装软件列表
brew update 更新Homebrew
brew outdated 显示可更新软件
brew cleanup [options] [formula] 清理旧版本软件 cleanup rediscleanup -s
brew config 显示配置信息
brew deps [options] formula 显示软件包依赖
brew tap [options] user/repo [URL] 添加一个第三方软件包仓库 brew tap beeftornado/rmtree
或列出所有已添加的第三方
软件包仓库
brew untap 删除已添加的第三方软件包仓库
brew services start 启动服务 brew services start redis
brew services start –all 启动所有可用服务 brew services start –all
brew services stop 停止服务 brew services stop redis
brew services restart 重启服务 brew services restart redis
brew services list 显示所有服务
brew services cleanup 清除无用服务

环境变量

临时设置环境通过在命令行执行

1

  export 变量名=变量值

永久设置环境变量需要编辑~/.bash.profile

1

  vi ~/.bash.profile

输入

1

  export 变量名=变量值

常用环境变量

变量 释义 示例
http_proxy 设置HTTP代理 export http_proxy=127.0.0.1:8087
all_proxy 设置SOCKS5代理 export all_proxy=127.0.0.1:1086
HOMEBREW_NO_AUTO_UPDATE 设置是否关闭自动更新Homebrew，关闭后Homebrew export HOMEBREW_NO_AUTO_UPDATE=true
将不会在执行brew install, brew upgrade 及brew tap时自动更新

brew rmtree

概述

rmtree用于卸载软件包以及该软件包的所有依赖

安装

1

  brew tap beeftornado/rmtree

使用示例

1

  brew rmtree emacs

brew cask

概述

brew用于安装无界面应用程序，而brew cask用于安装图形界面应用程序

常用命令

命令 释义 示例
brew cask info 显示软件包信息 brew cask info emacs
brew cask install 安装软件包 brew cask install emacs
brew cask uninstall 卸载软件包 brew cask uninstall emacs
brew cask list 列出已安装软件包 brew cask list
brew cask upgrade 升级软件包 brew cask upgrade emacs
brew cask outdated 列出可更新软件包

安装配置Redis

安装

1

  brew install redis

修改Redis密码

1

  vi /usr/local/etc/redis.conf

1

  # requirepass foobared

改为：

1

  requirepass 123456

123456即Redis密码

启动Redis服务

1

  brew services start redis

登录Redis

1

  redis-cli

验证密码：

1

  auth 123456

安装配置MySQL

安装

1

  brew install mysql

启动MySQL服务

1

  brew services mysql

安全设置

安装完后，root用户默认无密码，需要运行mysql_secure_installation设置root用户密码及其他安全设置

1

  mysql_secure_installation

根据提示设置密码，远程登录等

Homebrew 软件安装目录

1

  /usr/local/Cellar

1

  /usr/local/bin
