文件操作

快捷键 释义
C-x C-f 打开文件
C-x C-s 保存文件
C-x C-w 存为新文件
C-x C-c 退出Emacs

窗口操作

快捷键 释义
C-x 0 关闭本窗口
C-x 1 只留下一个窗口
C-x 2 垂直均分窗口
C-x 3 水平均分窗口
C-x o 切换到别的窗口
C-x s 保存所有窗口的缓冲
C-x b 选择当前窗口的缓冲区
C-M v 另外一个窗口向下翻页
C-M-Shift v 另外一个窗口向上翻页

缓冲区列表操作

快捷键 释义
C-x C-b 打开缓冲区列表
C-x k 关闭缓冲区

配置

vi ~/.emacs

(require 'package)

;设置中国elpa begin
(setq package-archives '(("gnu"   . "http://elpa.emacs-china.org/gnu/")
             ("melpa" . "http://elpa.emacs-china.org/melpa/")))
;设置中国elpa end

(package-initialize)

;设置smex begin
(require 'smex)
(smex-initialize)
(global-set-key (kbd "M-x") 'smex)
(global-set-key (kbd "M-X") 'smex-major-mode-commands)
(global-set-key (kbd "C-c C-c M-x") 'execute-extended-command)
;设置smex end

;设置evil begin
(require 'evil)
(evil-mode 1)
;设置evil end

;设置evil-org
(require 'evil-org)

;设置dired begin
(global-set-key (kbd "<f8>")
    (lambda ()
        (interactive)
            (dired "~/Nutstore Files/Note")))
;设置dired end

;设置org-page begin
(require 'org-page)
(setq op/repository-directory "/Users/jason/github/JasonChanBest.github.io")
(setq op/site-domain "https://jasonchanbest.github.io/")
(setq op/site-main-title "Jason Chan's Blog")
(setq op/site-sub-title "你必须非常努力才能看起来毫不费力")
;设置org-page end

;设置plantuml begin
(add-to-list
  'org-src-lang-modes '("plantuml" . plantuml))

(setq org-plantuml-jar-path
      (expand-file-name "/Users/jason/.emacs.d/plantuml.jar"))

(org-babel-do-load-languages
  'org-babel-load-languages
  '(
    (plantuml . t)
    ))
;设置plantuml end

;设置cnfonts begin
(require 'cnfonts)
(cnfonts-enable)
(cnfonts-set-spacemacs-fallback-fonts)
;设置cnfonts end

;设置主题
(load-theme 'atom-dark t)

;设置Org Mode以缩进模式显示
(setq org-startup-indented t)

;设置保存不备份临时文件
(setq make-backup-files nil)

;设置窗口最大化begin
(add-to-list 'default-frame-alist '(fullscreen . maximized))

;设置启动不显示欢迎页
(setq inhibit-startup-screen t)

;设置隐藏滚动条
(scroll-bar-mode -1)

;设置自动换行
(add-hook 'org-mode-hook (lambda () (setq truncate-lines nil)))

;设置代码高亮
(setq org-src-fontify-natively t)

;设置默认读入文件编码
(prefer-coding-system 'utf-8)

;设置写入文件编码
(setq default-buffer-file-coding-system 'utf-8)

;TODO状态改为DONE时自动插入时间
(setq org-log-done 'time)

;导出HTML时不要执行代码块
(setq org-export-babel-evaluate nil)

;导出HTML时不导出drawers
(setq org-export-with-drawers nil)

抽屉

有时候你想记录下一项事务的某些信息,但是这些信息在一般情况下是没必要显示出来的。为了实现这个目的,Org mode提供了drawers功能，即在标题上按 抽屉里的信息不会显示出来，只有在抽屉上按 时才会显示出来。

快捷键

快捷键 命令 释义
C-c C-x d org-insert-drawer 插入抽屉

待办事项

任何以TODO开头的标题都被认为是TODO事项，例如：

*** TODO Write letter to Sam Fortune

快捷键

快捷键 命令 释义
C-c C-t org-todo TODO 状态从 (unmarked) -> TODO -> DONE 循环
C-u C-c C-t 使用补全方式或快速选择界面(需要设置)来选择特定的TODO关键字
S-RIGHT 循环TODO状态
S-LEFT 循环TODO状态
C-c / t org-show-todo-tree 折叠起整个buffer，只显示TODO事项
C-c a t org-todo-list 显示全局的TODO列列表
S-M-RET org-insert-todo-heading 在当前TODO事项下面增加一个新的TODO事项
C-c , 设置TODO事项优先级
S-UP 提升TODO事项优先级
S-DOWN 降低TODO事项优先级
C-c C-c 更新任务进度

配置

  • 扩充TODO关键字

    默认情况下，TODO事项只有TODO和DONE个关键字，可以通过配置扩充关键字，如：

    (setq org-todo-keywords
  '((sequence "TODO" "FEEDBACK" "VERIFY" "|" "DONE" "DELEGATED")))

    “|”符号左边为未完结状态关键字，右边为完结状态关键字

    • 记录TODO事项完结时间

    当TODO事项状态切换为完结时自动插入时间：

    (setq org-log-done 'time)

  • 在TODO事项完结时记录附加信息

    (setq org-log-done 'note)
    • 优先级

    默认情况下,Org-mode支持从高到低三个优先级,分别表示为`A`,`B`,`C`. 如果某个任务没有分配优先级,则被认为是`B`优先级. 为任务分配优先级的意义仅仅在于在agenda视图中可以依照优先级对任务进行排序

    • 划分子任务

    在TODO事项下创建子TODO事项树来表达分层关系，在父TODO事项标题任何位置插入“[/]”或“[%]”来显示子TODO事项的完成情况，如：

    * Organize Party [33%]
** TODO Call people [1/2]
*** TODO Peter
*** DONE Sarah
** TODO Buy food
** DONE Talk to neighbor

    当所有子TODO事项状态都设置为完结后，父TODO事项状态也自动设置为完结：

    (defun org-summary-todo (n-done n-not-done)
  "Switch entry to DONE when all subentries are done, to TODO otherwise."
  (let (org-log-done org-log-states)   ; turn off logging
    (org-todo (if (= n-not-done 0) "DONE" "TODO"))))
(add-hook 'org-after-todo-statistics-hook 'org-summary-todo)

    checkbox列表

    当子checkbox列表状态为完结时，父checkbox状态会自动切换为完结示例：

    * TODO Organize party [2/4]
- [-] call people [1/3]
- [ ] Peter
- [X] Sarah
- [ ] Sam
- [X] order food
- [ ] think about what music to play
- [X] talk to the neighbors

快捷键

快捷键 命令 释义
C-c C-c org-toggle-checkbox 循环切换checkbox状态

标签

每个标题都可以包含多个tag; 这些tag应该放在标题的末尾位置. tag通常由字母,汉字,数字,-和@组成. tag的前后必须被冒号包括，示例：

* Meeting with the French group      :work:
** Summary by Frank                  :boss:notes:
*** TODO Prepare slides for him      :action:

快捷键

快捷键 命令 释义
C-c C-q org-set-tags-command 为当前标题输入新的标签
C-c C-c org-set-tags-command 光标位于标题上时，为当前标题输入新的标签

快捷键

快捷键 命令 释义
C-c . org-time-stamp 光标位于时间戳上时，修改时间戳
否则插入新的时间戳，如果两次执行该命令，则插入时间段
C-c ! org-time-stamp-inactive 类似C-c .命令，但插入的是一个无效的时间戳
C-u C-c . 类似C-c .命令，但插入的是带日期和时间的时间戳，默认的时间

C-u C-c ! 类似C-c !命令，但插入的是带日期和时间的无效时间戳，默认的时间

C-c C-c 更正时间戳,如果星期数错了或者没有,就会修正或者插入正确的星期数
C-c < org-date-from-calendar 光标位于时间戳上时，修改时间戳为当前时间，否则直接插入当前时间戳

S-DOWN org-timestamp-down-down 减少时间戳光标所在部分的值

杂项

自动完成

快捷键为：

M-TAB
  • 在标题开头，自动完成 TODO 关键字
  • 在“\”后面，自动完成TeX符号
  • 在“*”后面，自动完成标题
  • 在“:”后面，自动完成标签
  • 在“[”后面，自动完成超链接
  • 在“#+”后面，自动完成相关关键字，如OPTIONS
  • 在“#+STARTUP:”后面，自动完成startup关键字

快捷模板

例如输入\可以快捷插入

#+BEGIN_SRC
#+END_SRC

q   #+BEGIN_QUOTE ... #+END_QUOTE
v   #+BEGIN_VERSE ... #+END_VERSE
c   #+BEGIN_CENTER ... #+END_CENTER
C   #+BEGIN_COMMENT ... #+END_COMMENT
l   #+BEGIN_EXPORT latex ... #+END_EXPORT
L   #+LATEX:
h   #+BEGIN_EXPORT html ... #+END_EXPORT
H   #+HTML:
a   #+BEGIN_EXPORT ascii ... #+END_EXPORT
A   #+ASCII:
i   #+INDEX: line
I   #+INCLUDE: line

Dired

Dired是Emacs自带的文件管理器

绑定快捷键

设置按用Dired打开指定目录：

vi .emacs

(global-set-key (kbd "<f8>")
    (lambda ()
    (interactive)
        (dired "~/Nutstore Files/Note")))
